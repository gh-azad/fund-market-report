# پایش رقابتی مدیران دارایی — داده فیپ‌ایران

گزارش تعاملی سهم بازار و خالص ارزش دارایی مدیران دارایی فعال در صندوق‌های سرمایه‌گذاری ایران،
بر مبنای داده فیپ‌ایران، از فروردین ۱۳۹۹. به‌روزرسانی ماهانه خودکار.

- `index.html` — گزارش تعاملی (ماهانه/فصلی، فیلتر نوع صندوق، حذف بازارگردانی، مقیاس‌های ریالی)
- `data.js` — دیتاست تجمیعی (تولیدشده توسط اسکریپت‌های پایپ‌لاین محلی)
